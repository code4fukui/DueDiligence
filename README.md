# DueDiligence

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

This is a standard procedure manual for intellectual property due diligence, forked by Code for Fukui from the original open-source project.

## Features

- Standardized procedure for intellectual property due diligence
- Covers analysis of value-creating technologies, brands, designs, contents, know-how, and trade secrets
- Investigates utilization and scope of usage for identified intellectual properties
- Surveys intellectual property-related disputes and risks of infringing third-party rights
- Examines governance structure for intellectual property management
- Provides guidance for value assessment

## Requirements

- pandoc
- make

## Usage

```bash
brew install pandoc make
make
```
