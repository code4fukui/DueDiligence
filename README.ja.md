# DueDiligence

これは、知的財産デューデリジェンスのための標準手順書で、Code for Fukuiがオープンソースプロジェクトから派生したものです。

## 機能

- 知的財産デューデリジェンスの標準化された手順
- 価値創造技術、ブランド、デザイン、コンテンツ、ノウハウ、営業秘密の分析
- 特定された知的財産の利用状況と利用可能範囲の調査
- 知財関連紛争と第三者権利侵害リスクの調査
- 知的財産管理のガバナンス体制の調査
- 知的財産の価値評価の指針提供

## 必要環境

- pandoc
- make

## 使い方

```bash
brew install pandoc make
make
```
