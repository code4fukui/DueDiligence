# DueDiligence


これは、オリジナルのオープンソースプロジェクトからCode for Fukuiによってフォークされた、知的財産デューデリジェンス標準手順書です。

## 特徴

- 知的財産デューデリジェンスの標準手順
- 価値源泉となる技術等、ブランド、デザイン、コンテンツ、ノウハウ、および営業秘密の分析を網羅
- 特定された知的財産の利用可能性・利用可能範囲の調査
- 知的財産関連紛争および第三者の権利を侵害するリスクの調査
- 知的財産管理のためのガバナンス体制の調査
- 価値評価のためのガイダンスを提供

## 要件

- pandoc
- make

## 使用方法

```bash
brew install pandoc make
make
```
